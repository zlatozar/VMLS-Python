## VMLS-Python

This is a companion in Python for the book:
_"Introduction to Abstract Algebra: Vectors, Matrices, and Least Squares"_ by Boyd and Vandenberghe. 

Which you can find here: http://vmls-book.stanford.edu/.

You can also find a Python companion book in this repository.

## Use 'uv' to manage project dependencies

1. Install [uv](https://github.com/astral-sh/uv).
2. Sync dependencies: `uv sync`
3. Install git hooks: `uv run pre-commit install`
